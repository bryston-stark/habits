# Habit Tracker (React + Tailwind, no backend)

A browser-only habit-tracking web app with:

- Daily habits
- Weekly calendar with green/red done/missed indicators
- Intensity tracking (0-5) per day (replacing binary yes/no)
- Streak tracking
- Badge milestones (7, 30, 100 days)
- Gamification via points per habit and total score
- Persistence using `localStorage`

## Run

Open `index.html` in a browser.

No backend and no build step required.
